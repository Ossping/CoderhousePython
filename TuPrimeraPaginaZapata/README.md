# TuPrimeraPaginaZapata

## Requisitos
- Django instalado
- Python 3.7+

## Funcionalidades
- Crear categorías, autores y posts.
- Buscar posts por título.

## Instrucciones
1. `python manage.py migrate`
2. `python manage.py runserver`
3. Navegar:
   - `/categoria/new/`
   - `/autor/new/`
   - `/post/new/`
   - `/buscar/`
